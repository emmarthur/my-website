# Skill Logos

This folder contains SVG logo images for all the skills displayed in the portfolio.

## Required Logo Files

Place the following SVG files in this directory:

### Programming Languages
- `python.svg`
- `cpp.svg` (for C/C++)
- `javascript.svg`
- `java.svg`
- `bash.svg`

### Frameworks & Libraries
- `bootstrap.svg`

### Systems
- `linux.svg`
- `network.svg`
- `os.svg`
- `sysadmin.svg`

### Development Tools
- `servicenow.svg`
- `blueprism.svg`
- `jira.svg`
- `sonarqube.svg`

### Databases
- `mysql.svg`
- `sql.svg`

### Cloud & DevOps
- `git.svg`
- `azuredevops.svg`
- `kubernetes.svg`
- `docker.svg`

### AI Tools
- `github.svg` (for GitHub Copilot)
- `microsoft.svg` (for Microsoft Copilot)
- `openai.svg` (for ChatGPT and OpenAI APIs)
- `anthropic.svg` (for Claude)
- `ai.svg` (for Prompt Engineering)

## Where to Get Logos

Recommended sources:
1. **Simple Icons**: https://simpleicons.org/ - Free SVG icons, search for the technology name
2. **Devicon**: https://devicon.dev/ - Developer icons
3. **Official Brand Assets**: Check the official websites of each technology

## Logo Requirements

- **Format**: SVG (preferred) or PNG
- **Size**: Recommended 512x512px or larger (will be scaled down)
- **Style**: Monochrome or colored (will work with both)
- **Naming**: Use exact filenames as listed above

## Notes

- If a logo file is missing, the component will show a fallback with the first letter of the skill name
- Tooltips will always show the full skill name on hover for accessibility
- All logos should be properly licensed for use
