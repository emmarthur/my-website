# Section Images

This folder contains images for each section of the website.

## Required Images:
- `career.png` or `career.svg` - Professional/career themed image
- `hobbies.png` or `hobbies.svg` - Hobbies/creative activities image
- `relationships.png` or `relationships.svg` - People/connections image
- `thoughts.png` or `thoughts.svg` - Reflection/thinking image
- `media.png` or `media.svg` - Media/videos/image content
- `portfolio.png` or `portfolio.svg` - Portfolio/resume image
- `education.png` or `education.svg` - Education/learning image
- `work.png` or `work.svg` - Work/office image
- `projects.png` or `projects.svg` - Projects/code image

## Image Guidelines:
- Recommended size: 64x64px to 128x128px
- Format: PNG (with transparency) or SVG
- Style: Should match the deep sea blue theme
- Should be simple, recognizable icons/images

## Using Unsplash API (Optional):
You can use the Unsplash API to fetch relevant images dynamically, or use static images placed in this folder.
